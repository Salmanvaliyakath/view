# Salman Portfolio Website - Node.js + Express

A modern personal portfolio website for an AI/ML Engineer and Generative AI specialist.

## Tech Stack

- Node.js
- Express.js
- EJS templates
- HTML/CSS/JavaScript

## How to Run

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Production Run

```bash
npm install --production
npm start
```

## Customize

Edit the portfolio object in `server.js` to update:

- Name and title
- Projects
- Skills
- Experience
- Contact links
- Metrics

## Suggested Deployment Options

- Render
- Railway
- DigitalOcean App Platform
- Azure App Service
- AWS Elastic Beanstalk
